Este proyecto fue realizad en el entorno de desarrollo eclipse y su ejecución se realiza a través de la consola de eclipse

Para tener este proyecto en eclipse siga los siguientes pasos

1.Descargue todas las carpetas que se encuentran dentro del repositorio GiuHub
2.Cree un nuevo proyecto en eclipse con el nombre de Reto_SofkU
3.Pegue todas las carpetas remplazando las actuales que se encuentran dentro del proyecto Reto_SofkU
4.Cree una base de datos local sql ( puede utilizar XAMPP ) para crear su BD local
5.Cree una nueva BD esta debe tener el nombre de (juego_sofka)

	NOTA
Tener en cuenta que si el nombre de la base de datos no coincide con el nombre presentado en el punto 5 el programa no va a 
funcionar y tendrá mensajes SQL donde se mencionara que la BD no se encuentra.

6.Seleccione la BD juego_sofka e importe la la BD que se encuentra en la carpeta (Complementos_Del_Proyecto) allí encontrara
un archivo .sql donde ya se encontrarán todas las tablas y los registros necesarios para el funcionamiento del programa.

7. Dentro del entorno de desarrollo eclipse en el proyecto ( Reto_SofkaU ) diríjase al paquete principal y ejecute la clase
Principal.java

	NOTA

Al momento de indagar durante el programa para ver su funcionamiento lea atentamente las instrucciones que aparecen en la 
consola de eclipse para tener un buen manejo de esta.

Cualquier duda o inquietud de acuerdo al funcionamiento del programa por favor hágala saber al correo donde se envió
el reto del repositorio GitHub

En la carpeta Complementos_Del_Proyecto se adjunto una imagen PNG donde se muestra el MER utilizado para el desarrollo del 
reto.
